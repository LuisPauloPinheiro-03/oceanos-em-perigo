# 🌊 Navegando para um Futuro Sustentável.

> Site educativo interativo sobre os impactos do descarte inadequado de resíduos nos oceanos — desenvolvido com HTML, CSS e JavaScript puro.

---

## 📌 Sobre o Projeto

**Navegando para um Futuro Sustentável.** é uma plataforma web informativa e interativa criada como projeto educacional, com foco no público jovem (crianças e adolescentes). O objetivo é conscientizar sobre a poluição dos oceanos de forma visual, acessível e divertida.

O projeto foi desenvolvido do zero utilizando apenas tecnologias front-end — sem frameworks, sem backend, sem dependências externas além das fontes web. Todo o código foi escrito e evoluído manualmente, com apoio de ferramentas de IA para auxiliar no aprendizado e na documentação.

---

## ⚠️ Nota sobre o Histórico de Commits
- **Durante o desenvolvimento, muitas das alterações foram salvas localmente entre dois dispositivos
(computador e notebook) sem versionamento ativo no Git. Ao tentar organizar e publicar
o histórico posteriormente, cometi um erro ao usar `git push --force` sem perceber que
isso substituiria todo o histórico remoto pelo estado de apenas uma versão.**

- **O resultado foi a perda do histórico de commits no repositório. Não sabia que isso era
possível acontecer dessa forma — e foi uma lição importante sobre como o `--force` deve
ser usado com cuidado.**

## ✨ Funcionalidades

- **Hero animado** com ondas em CSS, partículas flutuantes e contadores animados
- **Cards de impacto** com dados reais sobre poluição oceânica
- **Seção "Você Sabia?"** com 6 cartões flip 3D clicáveis
- **Oceano Vivo** — cena animada alternando entre oceano limpo e poluído
- **Gráfico de barras** animado com dados sobre a crise ambiental
- **Mini-jogo drag-and-drop** "Salve o Oceano" — arraste itens para a lixeira ou o oceano
- **Quiz interativo** com 5 perguntas, feedback educativo e placar
- **Seção de ações** com 6 atitudes práticas que qualquer pessoa pode adotar
- **Calculadora de impacto coletivo** com slider interativo
- **Menu mobile responsivo** com drawer lateral animado
- Scroll reveal, acessibilidade com `aria-*` e suporte a teclado

## 🌍 Objetivo Educacional

Este projeto foi desenvolvido com dois objetivos paralelos:

1. **Aprender** lógica de programação front-end na prática — HTML semântico, CSS organizado com design tokens, JavaScript modular com padrão IIFE e boas práticas de acessibilidade.

2. **Conscientizar** — informar crianças e adolescentes sobre os impactos reais da poluição nos oceanos, usando recursos interativos para tornar o aprendizado mais engajante.
